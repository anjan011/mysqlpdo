GetArray()

GetArray(string  $sql, array  $bind = array(), array  $pdoOptions = array()) : array

Executes a SELECT query and gets first row of result as array

This function works similarly as GetArrayList taking sql query and bound parameter values (if any). The difference is it only returns the first row. Use this when you are looking for just one record.

The array fetch mode can be set using SetArrayFetchMode() to be associative, numeric or both.

Parameters

string $sql

The sql statement. It will be prepared
array $bind

The parameters to be bound (if any)
array $pdoOptions

Extra PDO driver options

Returns

array

Examples

<?php

    $pdo = MySqlPDO::GetInstance();

    $sql = "select name,email from users where email = ?";

    $bind = array("anjan011@gmail.com");

    $row = $pdo->GetArray($sql,$bind);

See also

SetArrayFetchMode()

GetArrayList()

GetArrayList(string  $sql, array  $bind = array(), array  $pdoOptions = array()) : array

Executes a SELECT query and gets result as list of array

Use this function to execute select queries and get result data as list of array. The sql statement is executed as prepared statement. the $bind param lets you to pass the values for parameters.

GetObjectList()

GetObjectList(string  $sql, array  $bind = array(), string  $className = 'stdClass', array  $ctorArgs = array(), array  $pdoOptions = array()) : array

Executes the sql query and get data as list of objects

Like GetArrayList() this function performs the select query, but instead of array, it returns a list of objects. If no class name is provided, stdClass is assumed.

PDO sets the class variables BEFORE calling the constructor. You can use this to further process the data returned if you need

Parameters

string $sql

The sql query to be used in prepared statement.
array $bind

The parameter values for the query.
string $className

The PHP class name. The class must be defined.
array $ctorArgs

If the class constructor requires parameters, pass here.
array $pdoOptions

Extra PDO driver options.

Returns

array

Examples

<?php

    class User {

        public $userName = null;
        public $userEmail = null;

        public $mailtoLink = null;

        public function __construct($createMailLink) {

            if($createMailLink) {
                $this->generateMailtoLink();
            }

        }

        public function generateMailtoLink() {

            $this->mailtoLink = "<a href='mailto:{$this->userEmail}'>{$this->userName}</a>";

        }

    }

    $pdo = MySqlPDO::GetInstance();

    $sql = "select userName,userEmail from User where userEmail like ?";

    $bind = array("%anjan011%");

    $results = $pdo->GetObjectList($sql,$bind,'User',array(true));

    /*
     Array
    (
        [0] => User Object
            (
                [userName] => Anjan Bhowmik
                [userEmail] => anjan011@gmail.com
                [mailtoLink] => <a href='mailto:anjan011@gmail.com'>Anjan Bhowmik</a>
            )

    )
     */

Query()

Query(string  $query, array  $bind = array(), array  $pdoOptions = array()) : \PDOStatement

Prepares a PDOStatement, calls it's execute() method and returns it.

This function takes your sql query and parameter values, executes that and instead of returning the results itself, it provides you with the PDOStatement object.

This will be quite useful in situations, where the resul tset is quite big and can take up a lot of memory. In this case you can use the PDOStatement and loop through the result set using PDOStatement::fetch() method.
